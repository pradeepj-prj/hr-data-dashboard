# HR Data Dashboard

Interactive Streamlit dashboard for exploring and visualizing synthetic HR data generated by [hr-data-generator](https://github.com/pradeepj-prj/HR-Data-Generator).

## Features

- **Overview Dashboard**: KPI metrics (headcount, avg salary, tenure, gender split), seniority distribution, employment type breakdown
- **Organization Views**: Interactive treemap and network graph showing manager hierarchy
- **Compensation Analytics**: Salary distribution, box plots by seniority/business unit, change reason breakdown
- **Performance Analytics**: Rating distribution, trends over time, heatmaps by business unit
- **Geographic Map**: Interactive Pydeck map showing employee distribution by location

## Installation

### Prerequisites
- Python 3.10+
- hr-data-generator library installed

### Setup

1. Clone this repository:
```bash
git clone <repo-url>
cd hr-data-dashboard
```

2. Create and activate virtual environment:
```bash
python -m venv .venv
source .venv/bin/activate  # On Windows: .venv\Scripts\activate
```

3. Install hr-data-generator from local path (or PyPI when available):
```bash
pip install -e ../HR-Data-Generator  # Local development
# or
pip install hr-data-generator  # From PyPI
```

4. Install dashboard dependencies:
```bash
pip install -e .
```

## Running the Dashboard

```bash
streamlit run src/hr_dashboard/app.py
```

The dashboard will open in your browser at `http://localhost:8501`.

## Filters

| Filter | Description | Type |
|--------|-------------|------|
| Employee Count | Number of employees to generate (10-500) | Slider |
| Business Unit | Engineering, Sales, Corporate | Multi-select |
| Seniority Level | 1 (Entry) through 5 (Executive) | Multi-select |
| Salary Range | Min/max annual salary filter | Range slider |
